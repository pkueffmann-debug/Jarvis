FAQ

Häufige Fragen

Ja. JARVIS läuft vollständig lokal auf deinem Mac. Keine Emails, keine Screen-Inhalte, keine Sprachaufnahmen verlassen dein Gerät. Das KI-Modell läuft lokal, powered by Apple Silicon.

JARVIS benötigt macOS 13.0 (Ventura) oder neuer. Für optimale Performance empfehlen wir Apple Silicon (M1–M4). Intel-Macs werden unterstützt, haben aber eingeschränkte KI-Performance.

JARVIS hört kontinuierlich auf "Hey JARVIS" — vollständig offline, direkt im Neural Engine deines Macs. Erst nach dem Wake Word wird JARVIS aktiv. Dein Mikrofon wird niemals zu externen Servern gestreamt.

Ja! Wir bieten eine 7-Tage-Geld-zurück-Garantie ohne Fragen. Wenn du nicht zufrieden bist, erstatten wir dir den vollen Betrag. Kontaktiere hello@daylens.dev.

Ja — JARVIS läuft auf macOS (Apple Silicon & Intel) und Windows 10/11 (x64). Beide Installer findest du oben auf dieser Seite oder direkt auf GitHub Releases. Mac-spezifische Features wie iMessage, Apple Notes und Reminders sind auf Windows naturgemäß nicht verfügbar — alles andere (Chat, Voice, Screen Awareness, Web-Tools) funktioniert identisch.

macOS: Lade die .dmg herunter, öffne sie und ziehe JARVIS in den Applications-Ordner. Beim ersten Start führt dich JARVIS durch die Berechtigungen (Accessibility, Screen Recording, Mikrofon) — unter 3 Minuten.

Windows: Lade JARVIS-Setup.exe herunter und führe sie aus. SmartScreen warnt vor unsignierten Installern — klicke „Weitere Informationen" → „Trotzdem ausführen". Wähle Installations-Ordner, fertig.
