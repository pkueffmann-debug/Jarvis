FAQ

Häufige Fragen

JARVIS läuft als native macOS-App auf deinem Computer. Die KI-Anfragen werden sicher über die Claude API (Anthropic) verarbeitet — keine Daten werden dauerhaft auf Servern gespeichert. Deine Emails, Termine und lokalen Dateien verlassen deinen Mac nicht ohne deine Anfrage.
Beim ersten Start führt JARVIS dich durch ein Onboarding: Mikrofon (Spracheingabe), Bildschirmaufnahme (Screen Awareness), Bedienungshilfen (App-Steuerung) und optional Kontakte und Festplattenzugriff. Alle Berechtigungen können einzeln gewählt werden.
